# Privacy Policy

**Last updated:** January 14, 2026

## Overview

This application ("AI Shorts Publisher") is a personal automation tool designed for publishing short-form video content to social media platforms including YouTube and Instagram.

## Data Collection

This application does not collect, store, or process personal data from third-party users. The application only accesses:

- **Social Media Account Information:** Used solely for authenticating and publishing content to accounts owned by the application operator.
- **Video Content:** Created and owned by the application operator for publishing purposes.

## Data Usage

All data accessed by this application is used exclusively for:
- Authenticating with social media platforms (YouTube, Instagram)
- Publishing video content to authorized accounts
- No data is shared with third parties

## Data Storage

- API credentials are stored securely and are not shared
- Generated content is stored locally and on authorized social media platforms only
- No user data is collected or stored from viewers of published content

## Third-Party Services

This application integrates with:
- Google/YouTube API Services
- Meta/Instagram Graph API
- Google Gemini AI (for content generation)
- ElevenLabs (for voice generation)

Each service has its own privacy policy governing their data practices.
